/*Написать программу, которая из имеющенгося массива строк формирует массив строк,
Длинна которого меньше либо равна 3 */

1. Сначала вводим через консоль длинну массива
2. Через цикл for заполняем массив строк
3. Через цикл for выводим полученный массив на печать
4. Через цикл for проверяем условие if длинна элемента массива <= 3 то печатаем этот элемент, если нет, то переходим к следующему элементу.
